# تاپل ها


## تمرین اول

برنامه ای بنویسید که یک index از کاربر بگیرد و آیتم نظیر آن را از تاپل زیر حذف کند.

`(1, 2, 3, 4, 5, 6, 7, 8, 9)`


## تمرین دوم

برنامه ای بنویسید که یک عدد و یک ایندکس از کاربر بگیرد و آن عدد را در ایندکس مورد نظر در تاپل زیر قرار دهد.

`(1, 2, 3, 4, 5, 6, 7, 8, 9)`
