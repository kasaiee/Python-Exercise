## تمرین اول

یک لیست به نام keys و یک لیست به نام values داریم که هر یک حاوی مقادیری هستند. برنامه ای بنویسید که از این دو لیست یک دیکشنری بسازید.

<div dir="ltr">
```
keys = ['fname', 'lname', 'age', 'phone']
    'age': 44,
values = ['Heshmat', 'kasaie', 44, '09377000000']

Output: {
    'fname': 'Heshmat',
    'lname': 'kasaie',
    'age': 44,
    'phone': '09377000000',
}
```
